Charts
------

Checkout the charts at
[matachi.github.io/steam-survey-stats](http://matachi.github.io/steam-survey-stats/)

Setup
-----

1. `virtualenv env`
2. `source env/bin/activate`
3. `pip install beautifulsoup4`
